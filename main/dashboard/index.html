Help & Support

Learn more about how TheThirdEye protects your browsing data.

How does local AI analysis work?

The extension extracts structural signals from web pages (like the number of hidden iframes or suspicious keyword presence) without logging keystrokes or inputs. These signals are scored heuristically and passed to the backend for AI inference.

Are my passwords sent to a server?

No. Passwords are hashed locally using SHA-1. Only the first 5 characters of the hash are queried against the HIBP database using k-Anonymity. This guarantees cryptographic safety.
