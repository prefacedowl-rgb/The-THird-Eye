What We Protect

Powerful Shields Active Right Now

Four battle-tested protection layers working silently in your browser — zero configuration needed.

Password Breach Detection

Silently checks if your password has appeared in known data breaches using the HaveIBeenPwned API. Your actual password never leaves your device — only the first 5 characters of a SHA-1 hash are sent, using k-Anonymity.

🔒 k-Anonymity Protected

Tracker Blocker

Blocks advertising and tracking networks at the network level using Chrome's Manifest V3 declarativeNetRequest API. Zero runtime overhead — rules execute natively in the browser engine before requests even leave.

⚡ Zero Performance Cost

WebRTC IP Leak Prevention

Forces all UDP traffic through a proxy so your real IP address can't be exposed via WebRTC — even if you're behind a VPN. Essential protection for privacy-conscious users.

🌐 VPN Leak Safe

Cookie Control Center

Inspect, manage, and delete every cookie a site has stored — directly from the extension popup. See cookie names, domains, SameSite attributes, and remove them with a single click.

🍪 Full Visibility

Security Header Analyser

Grades every page you visit on its response headers — CSP, HSTS, X-Frame-Options, and more. Get an instant A–F report and understand exactly what each header means for your safety.

📊 Instant A–F Grade

Full-Screen Security Dashboard

A beautiful analytics dashboard showing your risk score, threat timeline, threats by source, breached domains, and blocked trackers — all populated from real local extension data.

📈 Real-Time Analytics
Simple Setup

Up and Running in 3 Steps

No accounts. No subscriptions. No cloud sync. Just install and stay protected.

1

Install the Extension

Open chrome://extensions, enable Developer Mode, click Load Unpacked, and select the TheThirdEye folder.

2

Pin & Open the Popup

Pin TheThirdEye to your Chrome toolbar. Click it to open the security popup — all 4 shields activate automatically. No API keys needed.

3

Browse Freely

TheThirdEye silently guards every tab. Password breaches, tracker blocks, cookie exposures, and header grades are captured in real time.

Under the Hood

Manifest V3

Built on Chrome's latest and most secure extension platform. declarativeNetRequest blocks trackers at engine level with zero JS overhead.

k-Anonymity
