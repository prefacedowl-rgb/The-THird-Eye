# TheThirdEye тАФ ML Model (eai/)

Local phishing-detection model trained on the **PhiUSIIL dataset** (235K URLs, 52 features).  
Serves predictions via **FastAPI on port 5000**, integrated as **Layer 3** in the Node.js backend.

---

## Architecture

```
Browser signals тЖТ Node.js :3000
    тЖТ Layer 1: Community Threat Lists  (URLhaus / OpenPhish)
    тЖТ Layer 2: Heuristics              (scorer.js)
    тЖТ Layer 3: ML Model  тЖР this service (:5000)
    тЖТ Layer 4: OpenRouter LLM          (fallback if ML offline)
```

---

## Quick Start

### 1. Install dependencies
```bash
cd eai
pip install -r requirements.txt
```

### 2. Train the model
```bash
python train.py
```
This will:
- Build `lookups/safe_domains.json` and `lookups/unsafe_names.json` from the CSV lookup tables
- Load `../dataset/PhiUSIIL_Phishing_URL_Dataset.csv` (235K rows)
- Train Logistic Regression + Random Forest, pick the better one
- Save artifacts to `models/` (model.joblib, scaler.joblib, feature_columns.json, training_medians.json, model_meta.json)

Expected output: **>90% accuracy** on the 20% test split.

### 3. Evaluate (optional)
```bash
python evaluate.py
```
Prints accuracy, ROC-AUC, confusion matrix, and top 15 feature importances.

### 4. Start the microservice
```bash
python serve.py
```
Runs at `http://localhost:5000`.

### 5. Start the Node.js backend (separate terminal)
```bash
cd ../backend
npm start
```

## API

### `GET /health`
```json
{
  "status": "ok",
  "model": "RandomForest",
  "accuracy": 0.9742,
  "safe_domains": 168
}
```

### `POST /predict-url`
**Request:**
```json
{
  "url": "https://example.com",
  "signals": {
    "isHTTPS": true,
    "passwordFields": 0,
    "hiddenIframes": 0,
    "externalScripts": 2,
    "redirectCount": 0,
    "creditCardFields": 0,
    "totalForms": 1,
    "popupCount": 0
  }
}
```
**Response:**
```json
{
  "score": 85,
  "verdict": "safe",
  "probability": 0.15,
  "source": "ml-model",
  "reasons": ["No significant threat indicators detected"],
  "category": "safe"
}
```

### `POST /predict` (raw features, for debugging)
```json
{ "features": { "URLLength": 45, "IsHTTPS": 1, ... } }
```

## Feature Mapping

The Node.js backend sends browser signals that map to PhiUSIIL dataset columns:

| Browser Signal      | PhiUSIIL Feature    |
|---------------------|---------------------|
| `url` (computed)    | URLLength, DomainLength, IsDomainIP, NoOfSubDomain, TLDLength, IsHTTPS, ... |
| `isHTTPS`           | IsHTTPS             |
| `passwordFields`    | HasPasswordField     |
| `hiddenIframes`     | NoOfiFrame          |
| `externalScripts`   | NoOfJS              |
| `totalForms > 0`    | HasSubmitButton      |
| `redirectCount`     | NoOfURLRedirect      |
| `creditCardFields`  | Pay                 |
| `popupCount`        | NoOfPopup           |
| remaining ~35 cols  | training medians (defaults) |
